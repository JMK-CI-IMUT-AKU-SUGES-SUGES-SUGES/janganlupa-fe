# JanganLupa.id

Frontend React untuk aplikasi manajemen tugas mahasiswa berdasarkan desain UI/UX.

## Tech stack

- React 19 + Vite
- React Router
- Tailwind CSS v4
- Lucide React (ikon)

## Halaman

| Route | Halaman |
|-------|---------|
| `/` | Landing |
| `/login` | Login |
| `/register` | Register |
| `/beranda` | Dashboard (Beranda) |
| `/daftar-tugas` | Kanban Daftar Tugas + modal detail |
| `/kalender` | Kalender & daftar tugas harian |
| `/profile` | Profil pengguna |

## Menjalankan

```bash
npm install
npm run dev
```

Buka http://localhost:5173

## Build production

```bash
npm run build
npm run preview
```
